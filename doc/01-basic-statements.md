# Basic Statements

```js
// expressions (NOTE, all parenthesis here are optional)
4 + 2 // -> 6
4 - 2 // -> 2
4 * 2 // -> 8
4 / 2 // (divide) -> 2
(4 + 2) / 2 // -> 3
4 + 2 / 2 // -> 5 (calculation order is similar to math)
4 > 2 // -> true
4 < 2 // -> false
4 >= 2 // -> true
4 <= 2 // -> false
4 == 2 // -> false
4 != 2 // (not equal to) -> true
2 == 2 // -> true
```

```js
let a // define a variable called a. You can think of it as if we prepared an empty box for holding values.

a = 3 // assign a value to variable a (this is an action, like putting a value into a box)

a = 4 // assign a new value 4 to a, note that a has nothing to do with 3 now

let b = 5 // a preferrable short hand, define a variable and prefill its value

a + b // 4 + 5 = 9
a * b // 4 * 5 = 20
a - b // 4 - 5 = -1 (calculations can be applied to variables in the same way as values)
```
